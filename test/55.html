Configuration Settings

Customize your LLM Web Search Plugin settings below to enhance your search experience.

Search Settings

The number of search results to return by default. Higher values may increase response time.
When enabled, the plugin will fetch the full content of search results. This provides more detailed information but may slow down responses.
The maximum number of characters to fetch from each search result. Larger values provide more context but use more resources.

Time Retrieval Settings

List of URLs to fetch current time information. Add one URL per line. The plugin will try each source in order until successful.

LLM Model Settings

Select the default LLM model to use for generating responses. Different models have different capabilities and resource requirements.
Controls the randomness of the LLM output. Higher values (e.g., 0.8) make output more random, lower values (e.g., 0.2) make it more deterministic.
The maximum number of tokens to generate in the LLM response. Higher values allow for longer responses but use more resources.

Advanced Settings

The User-Agent string sent with HTTP requests. Some websites may block requests with certain user agents.
When enabled, the plugin will log detailed information about requests and responses. Useful for debugging but may increase log size.
